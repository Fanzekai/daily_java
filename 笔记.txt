2.== null是判断map是否为null（即是否new分配空间，和其中的键值对没关系），若没有内容则true，否则false

3.在Java中，双引号""表示一个空的字符串字面量。这意味着它表示一个长度为0的字符串，即不包含任何字符的字符串。
4.null.equals(“”) 和 "".equals(null) 这两种写法的区别在于，前者会发生空指针异常。
5.    //str != null && !"".equals(str.trim()) && !"null".equals(str.trim());
String str=null;
这行代码声明了一个名为"str"的字符串变量，并将其初始化为null。这意味着"str"变量当前不引用任何实际的字符串对象。
在这种情况下，如果你尝试使用"str"变量进行任何操作，比如调用它的方法或者访问它的属性，就会导致空指针异常.

String str = "null";
在这行代码中， String str = "null"; 表示创建了一个新的字符串对象，并将其内容设置为 "null"。
这并不是将字符串变量 str 设置为 null，而是将其设置为一个包含 "null" 这个字面值的字符串。
